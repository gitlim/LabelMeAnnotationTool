In this task, your job is to place 3D bounding boxes around the 10 largest objects in the image. If there are less than 10 objects in the image, just label as many as you can.

(b) Result example

The instructions below tell you how to add and place a box around an object in the image. Remember, your goal is not to label only one object, but the 10 largest in the image, or at least as many as possible. WARNING: The placement of the bounding box needs to be precise. It should take around 20 seconds per object - PLEASE DO NOT RUSH.

Adding, selecting, and removing boxes:
  • Click the "Add" button on the upper left
  • Enter the name of the object - it should be short, preferably one to three words
  • After you enter the name, a red link should appear in the upper right corner. Red indicates that the object is selected for editing. You can click a blue link to select that object for editing
  • If an object is selected, you can click the "Remove" button on the upper left of the screen to delete it.
  • You can double click a link to edit the object's name


Below are three main steps to go through for editing a 3D box. Please read the following instructions very carefully.

The goal is to place the box around the object so that it as similar as possible to the object's location, orientation, and dimensions.

(1) Object Position and Rotation

The goal for this step is to place the box so that its position and rotation are the same as that of the indicated object.

Object Position
  1. Place your mouse over the yellow 3D bounding box.
  2. Click and drag it to where the object is in the image.
  3. Position the box so that the center of its bottom is the same as the location of where the object is resting.
Note: Please only label objects that are on another object or the floor. Don't label objects without support objects (for example, a light hanging from the ceiling).

Placing the box in its initial location

Object Rotation
  • Click and drag anywhere on the image that is not the selected box to rotate the box.
  • If the object is rectangular, you can tell if the rotation is correct by making sure that the sides of the 3D box are parallel to the sides of the object.

(2) Editing Bounding Box Dimensions

The goal of the step is to edit the dimensions of the bounding box so that they match the width, length, and height of the object exactly.
  1. Place your mouse over one of the red knobs on the 3D box. The sphere should turn blue.
  2. Then, click and drag the knob to change the dimensions of the box in that direction.
  3. Note: If it looks like the 3D box is distorted compared to the other objects in the image and it is impossible to accurately label an object, click the "orientation is incorrect" button. You will then be directed to the next image.

(3) Refinement

This is the final and most important step - refinement of everything that was covered in the first two steps. Make adjustments in the same way you did in the previous steps (by moving the box, rotating the box, or by changing its scale). Below are some tips for places to look to see if the box has been placed and scaled correctly.

Tips:
  • Navigation: There is a "nav" (short for "navigation") button on the upper left of the screen. Click it to turn on navigation mode. You can then click and drag anywhere on the screen to see the 3D box from a different angle. Use this tool to judge whether or not the dimension of the 3D box seem correct compared to the actual object. This is important because sometimes the box may look correct from one angle, but actually have the wrong dimensions. Click the button again to return to the normal mode.

  • Rectangular objects: Make sure the edges of the bounding box are aligned with the edges of the object and that the corners match the corners of the object.

  • Irregular objects: Do the best you can by first making sure that the position of the box is correct. Then, try to enclose the object with the smallest box possible. However, all of the object should look like it's inside the box. Use the navigation mode to help you.
