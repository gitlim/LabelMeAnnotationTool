The instructions below describe how to assign the correct support object and height for an object. WARNING: Assigning the proper support object and height should take about 10 seconds per box - PLEASE DO NOT RUSH.


There are two main steps for each box: assigning the proper support object and adjusting the height.

(1) Labeling the Support Object

The goal for this step is to assign the object in the image to the object that it is resting on. For example, chair or piece of furniture that is resting on the ground would be assigned to the groundplane, and a lamp resting on a desk would be assigned to the desk.

Labeling the Support Object
  1. On the right hand side are a list of links. If you place your mouse over a link, the corresponding box will turn red
  2. Find the link corresponding to the support object of the object that you want to assign.
  3. Click and drag the link of the object that you want to assign and drop it on the link for the support object.
  4. The list should update and show that the assigned object is now a child of the support object.
Note: If the object is hanging "in the air" (attached to a wall, hanging from the ceiling, etc), please do NOT assign it to a support object.



The cup in this image has been assigned to the coffee table that it is on. For this image, if there were a box around the TV, you would NOT assign it to anything because it is hanging on the wall and not resting on an object.

(2) Adjusting the Object Height

The goal of the object is to adjust the height of the object's position so that it is correct. You will only have to do this for some objects - objects that:
  1. Are supported by another object that is not the ground.
  2. Have a position that is below the top of the box of the support object.
For example, an object might be resting on the seat of a sofa. The box for the sofa would include the entire sofa, including the sofa's back. The object lying on the seat of the sofa would then be below the top of the box for the sofa.

Adjusting the object's height
  1. Make sure that the object has already been assigned to a support object.
  2. Click on the link for the object to select it. The link will then stay red and the corresponding box will turn yellow
  3. You will now see a plane grid that represents the support plane for that object. Click and drag the plane to move adjust its height.
  4. You will see blue cubes appear where the plane intersects the boxes of other objects, including the selected object's support object. Use these indicators to determine the proper height for the object's position

Note: While adjusting the height of the box's position, you will not see the box move. This is because the code automatically calculates the new position and scale of the cube so that it looks the same regardless of the height. If you don't understand what this means, don't worry about it - it's not important.
